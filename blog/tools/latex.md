# Latex

> 记录一下平常写latex公式时常用的命令

##### 希腊字母

> 常见希腊字母写法

| $\alpha$ | `\alpha` | $\beta$ | `\beta` | $\gamma$ | `\gamma` |
| -------- | -------- | ------- | ------- | -------- | -------- |
|          |          |         |         |          |          |
|          |          |         |         |          |          |

##### 数学公式

> 常见数学公式符号写法

| $\hat{x}$ | `\hat{x}` | $\bar{x}$ | `\bar{x}` | $\overline{x+y}$ | `\overline{x+y}` |
| --------- | --------- | --------- | --------- | ---------------- | ---------------- |
| $x_1$     | `\x_1`    | $x^1$     | `\x^1`    | $x_{1, 2}$       | `\x_{1, 2}`      |
|           |           |           |           |                  |                  |
